Flex Box Adventure

Era uma vez um Rei Arthur. Ele governou seu reino de forma justa. Mas Arthur tinha um problema. Ele era uma pessoa muito ingênua. Então, em um dia ensolarado, três irmãos alquimistas ofereceram a Arthur para trocar todas as suas moedas de ouro por moedas feitas de um novo metal mais valioso que eles haviam inventado - Bit Coins.

Arthur acreditou neles e deu-lhes todo o seu ouro. Os irmãos pegaram o ouro e prometeram devolver as moedas a Arthur em sete dias.

Sete dias se passaram. Os irmãos não apareceram. Arthur percebeu que havia sido enganado. Ele está com raiva e pretende se vingar deles. Vamos ajudá-lo a fazer isso com nossa arma - CSS Flex Box!

We made this game for You

1. Você costuma tropeçar e tentar descobrir qual combinação de propriedades do Flex Box faz com que o navegador faça o que você deseja.

2. Você deseja criar layouts de web complexos sem olhar constantemente para a página da web após cada pressionamento de Cmd / Ctrl + S no editor de código.

3. Você tentou aprender o Flex Box com tutoriais em vídeo e artigos, mas ainda não entendeu completamente como algumas partes dele funcionam.

4. Ou, se você é um mestre em CSS Flex Box, temos algo interessante e para você também (leia mais).
