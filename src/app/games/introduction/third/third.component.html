Class Binding:


Para criar a ligação de classe única, use o Prefix class seguida do ponto e o nome da classe CSS - por exemplo, [Class.success]=\"expressão\". O Angular adiciona a classe quando a expressão ligada é verdadeira, e remove a classe quando a expressão é falsa.
