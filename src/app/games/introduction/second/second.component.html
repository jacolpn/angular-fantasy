Property Binding:


A vinculação de propriedades no Angular ajuda a definir valores para propriedades de elementos ou diretivas HTML. Use a associação de propriedade para fazer coisas como a funcionalidade do botão de alternância, definir caminhos programaticamente e compartilhar valores entre os componentes.


Para vincular à propriedade de um elemento, coloque-o entre colchetes, [], que identifica a propriedade como uma propriedade de destino. Uma propriedade de destino é a propriedade DOM à qual você deseja atribuir um valor.


Ex: A propriedade target no código a seguir é a propriedade placeholder do elemento de input: {{ textHelp }}
