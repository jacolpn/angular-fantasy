Interpolation:


A interpolação de texto permite incorporar valores de string dinâmicos em seus modelos HTML. Use a interpolação para alterar dinamicamente o que aparece em uma visualização do aplicativo, como a exibição de uma saudação personalizada que inclui um "Olá Mundo".


Ex: Declare uma variavel com o nome "texto" e utilize a interpolação escrevendo "{{ textCorrectHtml }}" no html.
